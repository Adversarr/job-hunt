## 🔍 使用指南

### 如何快速查找问题？

1. **按章节浏览**：从上方章节索引进入对应主题域
2. **按标签检索**：使用标签过滤相关文档
3. **按公司检索**：查看 `#字节`、`#阿里` 等标签下的面试题

### 如何使用答题卡片？

1. **一句话结论**：面试前 15 秒快速复习
2. **核心定义/公式**：确保基础概念准确
3. **为什么**：深入理解原理，应对追问
4. **怎么做**：准备具体的工程实践细节
5. **权衡分析**：展示系统性思维
6. **高频追问**：提前准备追问场景
7. **常见错误**：避免面试中的典型坑

### 如何准备面试？

1. **基础必考**：完成第 1、2、3 章所有文档
2. **进阶深入**：完成第 4、5、6、7 章核心文档
3. **工程实践**：完成第 9、10 章推理优化
4. **项目经历**：按第 14 章框架整理项目
